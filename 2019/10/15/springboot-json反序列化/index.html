    springboot
    创建时间:2019-10-15 22:25
    字数:245 阅读:

    Springboot 中怎样将返回的json报文反序列化为一个对象呢？

    1、在POJO中添加json注释（该注释可用于类或者成员）

    2、实现反序列化类

    3、反序列化

    JSONObject obj = new JSONObject();

obj.put("ID","2");
obj.put("NAME","OO");
obj.put("SCORE","23.2323");
obj.put("MALE","true");
obj.put("RATING","A");

ObjectMapper mapper = new ObjectMapper();
User user = mapper.readValue(obj.toJSONString(),User.class);
